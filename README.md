# designpad
This is a hackpad that I made for the hackclub/stardance project, so I can more easily design my pcbs and other things on CAD softwares, it has an OLED screen, a 3x3 matrix , and a rotary encoder!

Why did I make this board??

I really like desiginig pcbs and other models in different CAD models, and with this hackpad, after assigning all of the shortucts, I think I can really improve on my skills
Also desiging this hackpad was a ton of fun!

The reason I didn't put a case for this pcb is cause I already have some plans on how to make one, not 3D printing

Features:
128x32 OLED display
EC11 Rotary Encoder(for zooming in ofc)
6x Cherry MX Switches
6x 1N4148 DO-35 Diodes
46x DSA Keycaps
